Welcome to Slugger!

This project was built during my first couple weeks with Flatiron School as a personal warm up before classes started each day. It is a part of a code-along tutorial I wrote to explain the basics of making a browser game for beginner coders.

  • Begin the game by pressing "P"
  • Use "WSAD" or the arrow keys to move the slug
  • Run into a snack to eat it
  • Avoid running into the edge and yourself
